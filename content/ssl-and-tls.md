---
title: SSL/TLS
---
SSL/TLS is a security protocol used to protect data sent over the internet.

It is commonly used in HTTPS to secure communication between a browser and a web server.

# SSL vs TLS
* SSL stands for Secure Sockets Layer.
* TLS stands for Transport Layer Security. TLS is the newer and more secure version of SSL.

# Main Purposes
* Encryption
    * Prevents others from reading the data.
* Authentication
    * Verifies that the server is really who it claims to be.
* Integrity
    * Ensures the data was not changed during transmission.

# Simple Example

When a website uses https://, TLS helps protect sensitive data like passwords, payment information, and personal details.

# Key Idea

SSL/TLS makes internet communication secure by encrypting data and verifying identities.
