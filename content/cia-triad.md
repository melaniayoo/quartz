---
title: CIA Triad
---
The CIA Triad is a basic cybersecurity model used to protect information.

CIA stands for:

# Confidentiality
* Only authorized people can access the information.

Example: using passwords, encryption, and access control to protect private data.

# Integrity
* Information should be accurate and not changed without permission.

Example: preventing someone from modifying a file or database record.

# Availability
* Systems and data should be accessible when needed.

Example: keeping a website online and preventing downtime from attacks or failures.
