---
title: Computer Network
---
A computer network is a group of connected devices that can communicate and share data with each other.

Examples of devices include computers, phones, servers, routers, and printers.

# Main Purpose

Computer networks allow devices to:

* share data
* access the internet
* communicate with servers
* use shared resources like printers or files

# Common Types
* LAN
    * A local network in a small area, such as a home, school, or office.
* WAN
    * A large network that connects devices across cities, countries, or the world.
* Internet
    * The largest global network connecting millions of devices.

# Important Components
* Router
    * Sends data between different networks.
* Switch
    * Connects devices within the same local network.
* Server
    * Provides services or data to other devices.
* Client
    * A device that requests data or services from a server.

# Simple Example

When you open a website, your computer sends a request through the network to a web server.
The server sends the website data back to your computer.
