---
title: Cryptography
---

Cryptography is the practice of protecting information by converting it into a secure form.

It is used to keep data private, verify identities, and protect data from being changed.

# Main Purposes
* Confidentiality
    * Keeps information secret from unauthorized people.
* Integrity
    * Makes sure data has not been changed.
* Authentication
    * Verifies the identity of users, systems, or websites.
* Non-repudiation
    * Prevents someone from denying that they performed an action.

# Common Concepts
* Encryption
    * Converts readable data into unreadable data.
* Decryption
    * Converts encrypted data back into readable data.
* Key
    * A secret value used to encrypt or decrypt data.
* Hashing
    * Creates a fixed-size value from data to check integrity.

# Simple Example

When a website uses HTTPS, cryptography helps protect passwords and personal information while they travel between the browser and the server.
